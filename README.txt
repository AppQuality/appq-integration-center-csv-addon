AppQ Integration Center - CSV Addon. Allow the Admin to export a list of Bugs as CSV.

== Description ==

AppQ Integration Center - CSV Addon is specifically designed to work with the AppQ Integration Center plugin.

The main idea of this addon is to allow the Admin to export CSV generated file with dynamically selected Bugs and Fields.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `appq-integration-center-jira-addon.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
